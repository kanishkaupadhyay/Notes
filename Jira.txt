- Fields hold the data
    a. Description

    c. Assignee

- Projects are container for Issues

# Team managed projects vs Company managed projects

- Team managed project do not require admin permissions to create.
- All entities lives inside the project.
- Things like issue types, status, resolutions etc.
- Project scope entities.
- No special permissions to create 
- Fast and easy to set up and maintain.

- A company managed projects require admin permissions to create.
- Entities can be shared across other company-managed projects.
